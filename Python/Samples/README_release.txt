The following sample applications are included in this directory:
- gtiSamplePic.py  
- gtiSampleVideo.py
- gtiSampleWebCam.py
- gtiTwoChipsVideo.py

Prior to running above applications, please first install the GTI libraries:
cd ../Lib/
pip install --user gtilib/
make -C ../../Samples/Sample classifylib
pip install --user gtiClassify/
cd -
