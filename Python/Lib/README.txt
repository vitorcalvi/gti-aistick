This directory contains two python 2.7 pip installation packages
 - gtilib - python wrapper of libGTILibrary.so
 - gtiClassify - wrapper for libGtiClassify.so, for Legacy 3.1 Samples application support only

To install for the current user:
  pip install --user gtilib/
  pip install --user gtiClassify/
To install to the system python 2.7 directory:
  sudo pip install gtilib/
  sudo pip install gtiClassify/

To uninstall for the current user:
  pip uninstall gtilib
  pip uninstall gtiClassify
To uninstall from the system python 2.7 directory:
  sudo pip uninstall gtilib
  sudo pip uninstall gtiClassify
