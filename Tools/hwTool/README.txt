Hardware interface test tool can be used to test all the GTI supported interface protocol and hardware.

GTI AI chips support three hardware interfaces:

EUSB
SDA (via FTDI USB3.0 bridge chip) 
PCIe

Following GTI hardware products can be tested by this tool:

EUSB USB dongle with GTI2801
Direct EUSB connection daughter board with GTI2801
FTDI USB dongle with GTI2801
FTDI USB dongle with GTI2803
PCIe card with GTI2801
PCIe card with GTI2803

SDK v4.2 and abover versions also supports GTI5801.

SDK v4.2 and abover versions supports building this tools to run on Android commandline.
Please refer to the contains under sub-directory "Android" for details.
